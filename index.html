Советы

  • Используйте колесо мыши для масштабирования
  • Удерживайте пробел для временного переключения на инструмент руки
  • Двойной клик по карточке для редактирования текста
  • Используйте Markdown для форматирования текста в карточках
  • Перетаскивайте карточки, удерживая левую кнопку мыши
  • Используйте мини-карту для быстрой навигации по большой доске
